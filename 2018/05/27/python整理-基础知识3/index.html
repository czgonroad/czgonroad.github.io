>for循环

>>Pythonfor循环可以遍历任何序列的项目，如一个列表或者一个字符串。
for iterating_var in sequence:
  statements(s)
>>流程图：

>>对序列元素遍历
>>>for element in [1,2,3]:
    print(element)
>>>for key in ex_dic.keys:
    print(ex_dic[key])
>>对序列索引遍历
>>>for index in range(len(ex_lis)):
    print(ex_lis[index])
>>>for index in range(len(str)):
    print(str[index])
range(),len()为python内置函数，range(len(seq))表示序列元素数量n，生成0到n-1的整数列表，步长默认为1.
